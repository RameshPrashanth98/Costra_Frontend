### SplashScreen.tsx — HIFI faithful elements

**Visual hierarchy (top → bottom):**
1. Eyebrow: "Cost of Living Tracker" — JetBrains Mono 0.6rem, #C8FF00, uppercase, letter-spacing 0.2em
2. Logo: 128px SVG — outer dark circle (#0A0A0B, subtle border #1C1C1F), inner lime disc (#C8FF00), blocky square "C" path in #050505
3. Wordmark: small 14px lime C icon + "COSTRA" Outfit 900 0.95rem #FAFAFA, letter-spacing 0.15em
4. Title: "Daily money control for real life in Sri Lanka." — Outfit 800, 1.5rem, lineHeight 1.2, #FAFAFA, maxWidth 280px
5. Body: "Track essentials..." — Outfit 300, 0.85rem, lineHeight 1.65, #A1A1AA, maxWidth 300px
6. flex-1 spacer
7. Loading label: "Preparing your dashboard" — JetBrains Mono 0.65rem, #52525B, letter-spacing 0.15em
8. Three loading dots: sequential splashDotPulse CSS animation; active dot #C8FF00 10px, inactive #2A2A20 8px
9. Footer: "Built for everyday families..." — Outfit 300, 0.72rem, lineHeight 1.6, #52525B

**Animations:**
- `splashFadeUp`: opacity 0→1, translateY 18px→0, cubic-bezier(0.16,1,0.3,1), 0.65s
- `splashScaleUp`: opacity 0→1, scale 0.92→1 + translateY 12px→0 — applied to logo at 100ms delay
- Stagger delays: eyebrow 0ms / logo 100ms / wordmark 250ms / title 400ms / body 550ms / loader 750ms / footer 900ms
- `splashDotPulse`: scale + opacity pulse, 1.5s infinite; delays 0/200/400ms create travelling-dot effect

**Navigation:** useEffect + setTimeout(2800ms) with `timerRef` cleanup on unmount via `useNavigate('/welcome')`

**Layout:** `min-h-[100dvh]` flex column; `paddingTop: max(env(safe-area-inset-top), 48px)`; `paddingBottom: max(env(safe-area-inset-bottom), 32px)`; horizontal 28px padding
