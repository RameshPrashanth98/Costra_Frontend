**Files:** `src/flows/onboarding/screens/OnboardingScreen.tsx`

**Action:** Replace placeholder with full carousel component. Structure:

```tsx
// Pseudo-structure
const slides: SlideData[] = [
  { id: 1, eyebrow: 'WELCOME TO COSTRA', hasLogo: true, title: '...', body: '...',
    card: { type: 'bullets', bullets: [...] }, hero: '/hifi-heroes/onboarding-1.png',
    cta: 'Continue', showSkip: true },
  { id: 2, eyebrow: 'FLEXIBLE DAILY CONTROL', title: '...', body: '...',
    card: { type: 'amount', label: 'MONEY LEFT TODAY', amount: 'Rs.1,480', caption: '...' },
    supportingText: '...', hero: '/hifi-heroes/onboarding-2.png', cta: 'Next' },
  { id: 3, title: '...', body: '...',
    card: { type: 'labeled-bullets', label: 'ONE TAP OR VOICE', heading: '...', bullets: [...] },
    hero: '/hifi-heroes/onboarding-3.png', cta: 'Next' },
  { id: 4, title: '...', body: '...',
    card: { type: 'labeled-bullets', label: 'SMART ALERTS', heading: '...', bullets: [...] },
    supportingText: '...', hero: '/hifi-heroes/onboarding-4.png', cta: 'Get Started' },
];

const [index, setIndex] = useState(0);
const [direction, setDirection] = useState<'fwd' | 'none'>('none');
const navigate = useNavigate();

const next = () => {
  if (index < 3) { setDirection('fwd'); setIndex(i => i + 1); }
  else { navigate('/register'); }
};
const skip = () => navigate('/register');
```

- Bottom chrome (fixed):

  - Primary CTA
  - Skip link (slide 1 only)

**Hero image cropping strategy:**

The HIFI PNGs contain the full mockup (393×852). The illustration portion sits roughly between y=55%-78% of the image. Use CSS:

```tsx
<div style={{
  width: '100%', aspectRatio: '16 / 9', // or specific height
  borderRadius: 16, overflow: 'hidden',
  backgroundImage: `url(${hero})`,
  backgroundSize: '100% auto',
  backgroundPosition: 'center 66%',
  backgroundRepeat: 'no-repeat',
}} />
```

This crops each PNG to show only the hero illustration region. Tweak `backgroundPosition` y% per slide if needed (slide 1 has logo at top, text middle, illustration lower-middle — may need different offset). Expected range: 60%-70% for all slides.

**Animations (CSS-only, no deps):**

1. **Entrance animation** (on mount and on slide change):
   - Each text element (eyebrow → title → body → card → hero → CTA) fades + rises (`animation: onboardFadeUp 0.5s cubic-bezier(0.16,1,0.3,1) both`) with staggered delays (0, 80, 160, 240, 320, 400ms)
   - Re-triggered on slide change by keying the slide container on `index`
2. **Slide transition**: wrap slide content in a div with `key={index}` so React unmounts + remounts, re-playing entrance keyframes
3. **Pagination dot animation**: active dot grows from 8px→10px with `transition: all 0.3s ease`; color crossfade
4. **Button press**: subtle scale-down on active (`:active { transform: scale(0.98) }`)
